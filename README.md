# Node.js MVC OOP Project

Basic REST API using Node.js, TypeScript, and MySQL.

## How to Setup
1. Clone the project
2. Run `npm install`
3. Import `oop_test.sql` to my MySQL
4. Change DB config in `src/config/db.ts`
5. Run `npm run dev`


## Features
- CRUD Users
- MVC Architecture


## API Endpoints
- GET /users
- POST /users
- Put /users/id
- Delete /users/id
